About The Website

This website focusses on providing users with probabilities of various health risks using medical datasets.

  • Explore individual prediction models for Heart Disease, Stroke and Diabetes
  • Discover our most advanced Multiple Disease Predictor that seamlessly integrates all features into one

The Project

The aim of the project was to build a predictor that was accurate as possible using the resources available online. Though not medically licensed to be a medical tool the parameters can always be updated when more accurate datasets become available. This is why features such as “Nearest Medical Professional near me” provide direct and helpful links to the NHS website for more information.

The website provides a journey into the evolution of the webpage from simple predictors to more complex ones.
