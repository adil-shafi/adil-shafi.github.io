Heart Disease Datatable (original)

This was the original dataset that was used for most of the javascript trial and error learning. It is a popular set of data that has been used globally for various machine-learning projects. The initial aim of the project was to create a disease predictor that took into account age, sex and location, which this dataset had. You can see in the Heart Disease Predictor these are the only inputs the user is able to put with the addition of smoking and drinking variables later added. The other attributes in the table were disregarded as it was discussed that the average user may not have knowledge of the other factors, and if they did it is most likely they have obtained this by visiting a medical professional making this predictor for them redundant.r
