Heart Disease Datatable (Modified)

This dataset is different to the one used in the hardcoded version

This dataset was taken from Kaggle which is a well-known and trusted source for gathering datasets. This particular dataset was chosen as its attributes were similar to the ones already implemented in previous versions of the predictor models (stroke and diabetes). The data here was great since it had a wide range of different attributes that could potentially create a more advanced predictor. As seen in the data table not only are the ages of the patients provided each patient has been further categorised into age groups; 30s, 40s etc. This feature appears throughout the dataset such as with cholesterol and bp levels.

Certain attributes were not included as it was unlikely users would have information such as their cholesterol or blood pressure on hand without first visiting a medical professional.

As mentioned this is a different dataset to the one uploaded to Heart Disease Datatable (Original). A different heart disease dataset was used as it seemed more advanced and interesting and more fitting for the MDP model.

Modifications to dataset

The orinal dataset set the patients sex to 1 and 0 the csv file was edited to change this to Male and Female to conform to the other datasets.
